# AI-Powered ATS Engine

An AI-based Resume Screening System built using Flask and NLP techniques.

## Features
- Resume PDF Upload
- Text Extraction using PyPDF2
- TF-IDF Vectorization
- Cosine Similarity Matching
- ATS-style Candidate Ranking

## Tech Stack
- Python
- Flask
- Scikit-learn
- SQLite
- PyPDF2

## How It Works
1. Upload resume (PDF)
2. Extract text
3. Compare with job description
4. Generate similarity score
